# ComeHere
本项目是和几个小伙伴学习vue的成果，在这里提供给大家用来学习vue。

*借鉴了某旅游网站的布局，不得用于商业用途。如涉及侵权请联系本人。q 594402100

clone之后，先npm install，用cnpm更好。

本地运行  npm run dev/ npm start。
